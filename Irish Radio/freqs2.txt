    Anologue Cell Phones (Eircell 088)



  "Thanks to "otheLLo_" for the following information"

  The system works like this,
  Eircell or the phone operator have a set range, 

       

    918MHz to 934MHz  and  936MHz to 947MHz



  When someone makes a call either to or from a mobile phone, the telephone company
  automatically allocates a random frequency for the call. For example the same 
  phone won't have the same frequency, but will have a different one allocated
  every time a call is iniciated.

  The steps are in 12.5MHz NFM. If you start scanning at 918MHz NFM in 
  12.5 I think you will find you encounter numerous calls.

  So basically the individual frequencies cannot all be listed 
  but a set range is appropriate.
