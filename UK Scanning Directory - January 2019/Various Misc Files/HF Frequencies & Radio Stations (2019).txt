HF Frequencies Database (kHz) 2019


Unlike our other frequency listings which are in MHz our HF Database is in kHz up 
to 10.000kHz (10MHz). 1000kHz = 1MHz, so for example RAF Rescue is on 5680.0 kHz 
which is the same as 5.680 MHz

For band plans (Aircraft, Broadcast and Amateur) Coastguard nets and other
detailed broadcast and utility station listings see further below.
